Propósito del sitio

Este sitio web fue creado con el fin de dar mas información sobre la sintaxis y selectores CSS para la interacción entre humanos y ordenadores. Tambien se mostrara como CSS ha ayudado la educación ya que por culpa de la pandemia del Covid-19 se han buscado formas para desarrollar objetos virtuales de aprendizaje. Y mostrare un video sobre un tema diferente que son las inecuaciones y resolvere un ejemplo claro de ello.
